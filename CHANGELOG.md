# Changelog

## v0.1.0 - 2026-07-20

- 发布首个无预装业务卡带的正式基座。
- 提供 CF-FARP `0.1` 至 `0.5` registry，当前推荐 `CF-FARP@0.5`。
- 实现结构化 Decision Envelope、显式 consume、交互暂停与恢复。
- 实现测试台、范围探针、数据链诊断和产物收集。
- 实现 Portable DLC 描述符、SHA-256 完整性校验、作用域工具注册和 JSON stdio worker。
- 实现 sandbox iframe 前端扩展宿主和通用 DLC payload 传递。
- 实现卡带导入、打包、卸载及资源所有权策略。
- 支持 OpenAI Chat Completions 与 Responses API，并提供文本和图片理解连通测试。
- 移除探索期内嵌的像素剧集、短视频、空间预演和系列 3D 领域实现。
